Rangkuman Git

membuat repository local dengan perintah "mkdir git"

membuat beberapa commit dengan perintah "git add" untuk membuat staged file dan "git commit" untuk commit file

mengenal fungsi "git log"

Branch Git

membuat branch baru dengan perintah branch namafile

barpindah ke branch baru dengan perintah "git checkout"

Melakukan merge antar branch dengan perintah "merge"
